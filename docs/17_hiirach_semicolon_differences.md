## 全体像

『；』は、カードヒーローの戦闘構造を別ゲームに置き換える案ではない。基本は Pro ルールを土台にし、対人戦の安定性、リアルカード運用、カード採用幅、試合時間、白黒マスター間の環境バランスを調整する派生ルールである。

大きな変更方針は次の 5 点。

- デッキと手札を増やし、序盤事故とドローソース依存を緩和する。
- スーパーカードを即時行動寄りに強化し、実戦投入しやすくする。
- `きあい`、`にげる`、`メイクカード` のような原作コマンドを整理し、リアルカード運用を軽くする。
- 弱いカードを底上げし、強すぎるカードや処理負荷の高いカードを弱体化/削除/再設計する。
- 40 枚デッキ、公開情報、シャッフル回数、山札切れ、大会時間など、物理対戦で問題になる部分を調整する。

## カードプール設計の差分

### 基本方針

- DS 版カードを原則ベースにするが、一部カードは削除、調整、復活、新規カード化される。
- 使われないカードを実戦圏に上げる一方、環境を壊すカードは弱体化または削除する。
- 物理対戦で処理が重いカード、公開情報管理が面倒なカード、シャッフル回数を増やすカードは、運用しやすい処理へ寄せる。
- 白黒マスターのバランスを崩すカードは、単体性能だけでなく環境影響を見て削除/再設計する。

### 初期不採用・調整困難とされたカード

entry-392 では、調整しても元イメージを保ったまま実戦圏へ上げ下げしにくいカードが整理されている。

| 氷帝コルドラ | コールドブレスはペナルティでは解除されず、かげ呪いのダメージでは解除される。 |
| グングニエル | 攻撃範囲は、移動後に自身の一列前の横一列を攻撃する `なぎはらい`。 |
| コスト計算順 | バイストーンのような倍率を先に計算し、その後に呪い/デスシープ等の増減を計算する。 |
| ヤミー | 相手から実際に奪えた分だけ味方ストーンが増える。 |
| デスシープ | 効果は重複する。 |
| ファントム | 特技使用時、載っていたストーンは新モンスターへ載せ替える。レベル差分の余剰ストーンは消滅する。 |

## 採用/不採用の判断軸

『；』のカード変更は単純な強化ではなく、次の判断軸で行われている。

- 弱すぎて採用されないカードは、コスト軽減、HP/火力調整、ドロー付与、役割明確化で実戦圏へ寄せる。
- 強すぎるカードは、コスト増、HP低下、即時打点の抑制、対象制限、削除で抑える。
- スーパーカード強化により、種モンスターを無視してスーパーだけを買う採用が強くなりすぎる場合は、アンノウンのように原作寄りへ戻す。
- 速攻を完全否定する回復/延命カード、試合時間を延ばすカードは弱体化対象になる。
- ランダム系は、山札下を覚える運用負荷を避け、公開後は捨札へ置く方向へ変更される。
- シャッフルを増やすカードは、カードサーチのように捨札処理へ寄せ、試合時間を短縮する。
- 白黒バランスに過剰な影響を持つカードは、エクスチェンジのように削除対象になる。

## このプロジェクトで扱う時の境界

『；』は、原作互換実装の正解ではなく、熟練プレイヤーによる改良案である。このプロジェクトでは次の扱いにする。

- 原作準拠プロトタイプへ、そのまま 40 枚デッキ、6 枚手札、きあい廃止を入れない。
- CPU AI 改善では、`なぜそのカードを強化/弱体化したか` を判断軸として参照する。
- カード効果実装では、公式仕様と『；』独自仕様を混同しない。
- 独自ルールを検討する場合でも、カードヒーローの戦闘ルール互換を壊さない範囲で別モードとして扱う。
