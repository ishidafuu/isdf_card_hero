# White Mirror Response Guard Loop

生成: 2026-06-24

## 目的

白対白で、終端盤面だけを見ると良く見えるが、相手ターンを返すと崩れる手順を避ける。
特に seed 56002 の「ヤンバルをLv2にして後衛に残し、返しでボムゾウに倒される」手順を対象にした。

## 実装

- 白ミラー時だけ、相手応答評価を行う根候補を拡張。
- 白ミラー時だけ、終端評価が同点または僅差なら相手応答後の盤面で比較。
- 残石1以下で、相手応答後に大きく崩れる終端手順を追加で減点。
- 返しで倒されるLv2以上の後衛を渡す終端盤面を減点。
- プレイヤー側AIの攻撃評価でも `pendingLevelUp` を評価用に解決し、レベルアップ後のHPと露出を評価。
- ウェイクアップで削り役を起こし、前衛フィニッシャーにレベルアップ撃破を渡せる形を加点。
- 後衛の削り攻撃が、前衛フィニッシャーのレベルアップ撃破に変換される形を加点。

## 監査結果

対象: `submission-pro-with-rare8-white-1339` vs `submission-pro-no-rare8-white-1377`

`docs/master_lab/results/2026-06-24_white_ai_terminal_plan_audit.md`

- selected response top1: 0/3 -> 1/3
- selected average response rank: 9.33 -> 7.33
- average response gap to best: 54 -> 3.3
- max response gap to best: 134 -> 10
- seed 56002: response gap 129 -> 0

seed 56002 は、召喚 -> ウェイクアップ -> ヤンバルで削り -> モーガンで撃破してLv2、という順序に変わった。
これにより、ヤンバルLv2を返しで献上する手順を避け、応答後1位の手順を選べるようになった。

## 実測

`docs/master_lab/results/2026-06-24_white_mirror_response_guard_benchmark.md`

- 白対白 2ゲーム、issue 0
- WPR は 1339/1377 とも 50%
- 平均ゲーム時間: 25.4秒
- 平均ターン時間: 1.2秒
- 最大ターン時間: 6.0秒

実プレイ想定の1分/ターンには十分収まる。

## 所感

今回の改善は、単純な勝率調整ではなく「次ターンに返されると崩れる盤面を渡さない」方向の改善。
終端盤面単体の順位は落ちたが、相手応答後のギャップは大きく縮んでいるため、白ミラーの読みとしては前進。

次に攻めるなら、今回残った seed 56000 の盾込み0石着地を対象にする。
ただし応答後ギャップは最大10まで縮んでいるため、先に中母数の白対白勝率と対黒への副作用確認を挟む方がよい。
